AI 图像风格迁移工作台
这是一个基于 Web 的 AI 图像风格迁移演示应用，允许用户上传图片并应用不同的艺术风格，通过调整参数实时预览风格化效果。

功能特点
图片上传：支持用户上传自定义图片作为风格迁移的基础内容
多种风格选择：提供多种预设艺术风格（波浪纹、呐喊、乌德尼等）
参数调整：可调整风格强度（0.0 - 1.0）以控制风格化程度
实时预览：即时显示风格化结果
响应式设计：适配不同屏幕尺寸的设备
技术栈
HTML5
CSS3 (Flexbox布局)
JavaScript (ES6+)
TensorFlow.js (用于AI模型处理)
Canvas API (用于图像处理)
使用说明
上传图片：点击"上传图片"按钮选择您想要风格化的图片
选择风格：在"艺术风格"区域点击不同的风格按钮
调整参数：使用"风格强度"滑块调整风格化程度
生成结果：点击"开始迁移"按钮生成风格化图片
保存结果：右键点击生成的图片可保存到本地
